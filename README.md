# Spectral Identifiability in Delayed Epidemic Surveillance

Code and manuscript files for:

**Delay distributions limit the identifiability of rapid variations in epidemics**  
Jingjing Tang, Bryan Wilder, and Roni Rosenfeld

The manuscript has been submitted to medRxiv. A preprint link will be added once it is posted.

## Structure

```text
code/       analysis and figure-generation scripts
config/     dataset configuration files
paper/      manuscript source, figures, and compiled PDF
```

Generated figures are used in `paper/figs/`.
